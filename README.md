🚀 Easy Service Platform
A full-stack web app that connects users with local service providers (AC repair, electrician, cleaning, etc.).
✨ Features
👤 User & Shop Owner Authentication (JWT)
🏪 Shop Registration with Services & Pricing
📦 Order Management & Worker Assignment
💬 Real-time Chat (Socket.io)
📊 Dashboard with analytics
🧑‍💻 Tech Stack
⚛️ React (Vite)
🟢 Node.js + 🚀 Express.js
🍃 MongoDB
🔐 JWT
⚡ Socket.io
